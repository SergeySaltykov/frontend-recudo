# frontend-recudo

Запуск локального сервера [http://localhost:8000](http://localhost:8000)
```sh
npm start
```

Сборка билда для продакшена
```sh
npm run build
```

Автоматическая компиляция изображений при сборке билда
пример : import img from '../../www/content/img/test.jpg';

Подключить eslint
